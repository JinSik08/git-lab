Comandos de terminal

Códigos que puede usar:

Códigos de terminal
Código Definición
pwd Mostrar la ruta del directorio actual
ls Lista los archivos y carpetas en el directorio actual
cd "ruta" Cambiar al directorio especificado
mkdir "nombre de carpeta" Crear un nuevo directorio
touch "archivo" Crear un nuevo archivo vacío
cp "origen" "destino" Copiar un archivo
rm "archivo" Eliminar un archivo del repositorio y del sistema.
mv "archivo" "nueva ubicación" Mover un archivo a otro directorio.
git log Mostrar el historial de commits.
cat "archivo" Mostrar el contenido de un archivo
nano "archivo" Abrir el editor de texto Nano
vim "archivo" Abrir el editor Vim
