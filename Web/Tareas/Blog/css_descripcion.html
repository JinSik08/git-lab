Descripción de CSS

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la apariencia y el formato de un documento HTML.

Con CSS, podemos modificar aspectos visuales como colores, fuentes, tamaños, márgenes, espaciados y posicionamiento de los elementos en una página.

Selectores, propiedades y valores en CSS

CSS trabaja mediante reglas de estilo, que se componen de selectores, propiedades y valores.

  • Selector: Define a qué elementos se aplicará el estilo.
  • Propiedad: Especifica la característica que se va a modificar.
  • Valor: Indica el ajuste o configuración de la propiedad.

Ejemplo de la regla:


    p {
        color: blue;
        font-size: 18px;
        text-align: center;
    }

En este caso,

  • p, es el selector que aplica a todos los párrafos,
  • color, font-size y text-align, son las propiedades y
  • blue, 18px y center, son los valores.

    • Tipos de selectores:

    1. Selector por tipo o etiqueta

    Esto selecciona todas las etiquetas de mismo tipo específico.

    Por ejemplo:

    
    p {
        width: 300px;
        padding: 20px;
        background-color: lightblue;
    }

    Sirve para aplicar estilos a todos los párrafos p.

    1. Selector por clase (.)

    Esto selecciona solamente los elementos que tiene la clase específica.

    Por ejemplo:

    
    .ejemplo {
        height: 200px;
        background-color: green;
        font-weight: bold;
    }

    Sirve para aplicar estilos a solamente a los elementos que tiene la clase ejemplo.

    1. Selector por identificador (#)

    Esto selecciona únicamente el elemento que tiene el id específico.

    Por ejemplo:

    
    #title {
        font-size: 22px;
        color: lightgrey;
        text-align: right;
    }

    Sirve para aplicar estilos a únicamente al elemento que tiene el id.

    Si resumimos:

    Diferencias entre Selectores por Tipo, Clase e Identificador
    Selector
    		 Símbolo Donde se aplica
    Tipo Nombre de etiqueta Todos los elementos de ese tipo como párrafos, body, etc.
    Clase .Nombre Todos los elementos con esa clase.
    Identificador #Nombre Un solo elemento único en la página.
